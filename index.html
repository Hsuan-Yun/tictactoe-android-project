About Tic-Tac-Toe｜

✓ 用戶設定
　更改用戶名稱、顯示註冊和簽到天數、每日簽到、顯示遊戲場數、顯示勝率和敗率、新增朋友

✓ Create 玩法
　建立私人房間並進入遊戲

✓ Join 玩法
　加入私人房間並進入遊戲

✓ Random 玩法
　建立或加入公開房間

✓ Single 玩法
　單機玩法

✓ 排行榜
　查看全部用戶排行榜、查看朋友排行榜

✗ 僅支援安卓11.0以上系統

Built With｜

MIT App Inventor
Google Firebase
